= Beispiel für eine StorageGRID S3 REST API-Sitzungsrichtlinie
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Verwenden Sie das folgende Beispiel, um eine StorageGRID -Sitzungsrichtlinie zu erstellen.



== Beispiel: Einrichten einer Sitzungsrichtlinie, die den Objektabruf ermöglicht

In diesem Beispiel darf der Sitzungsbenutzer nur Objekte aus bucket1 abrufen. Alle anderen Aktionen sind implizit untersagt, mit Ausnahme von StorageGRID-spezifischen Aktionen, wie der Verwendung der link:../s3/use-access-policies.html#use-putoverwriteobject-permission["s3:PutOverwrite Object"] Berechtigung. Die Sitzungsrichtlinie kann beim Aufruf der AssumeRole API als JSON-Datei angegeben werden.

[listing]
----
{
  "Statement": [
    {
      "Action": "s3:GetObject",
      "Effect": "Allow",
      "Resource": "arn:aws:s3:::bucket1/*"
    }
  ]
}
----
